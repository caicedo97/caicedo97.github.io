# Portafolio de Jonatan Cotacio

Portafolio profesional estático, preparado para GitHub Pages y construido únicamente con HTML, CSS y JavaScript.

## Uso local

Abre `index.html` directamente o sirve la carpeta con cualquier servidor HTTP estático.

## Publicación

En GitHub, activa **Settings → Pages → Deploy from a branch** y selecciona la rama principal y la carpeta raíz.
